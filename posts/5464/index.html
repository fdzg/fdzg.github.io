ROS设置

Created2020-02-18|Updated2021-02-14|TECHNOLOGY - Ros
|Post View:

  1. 安装后初步设置

    • 1.1 安装后登录，用户名默认：admin；密码默认为空。
    • 1.2 常用命令：
      • 1.2.1 查看授权信息： system license print ;
      • 1.2.2 查看网口识别情况：interface print ;
      • 1.2.3 查看ros后台IP地址： ip export ; #官方ros系统后台地址为空
      • 1.2.4 恢复出厂设置：system reset;
      • 1.2.5 设置ros后台ip地址及网口：ip address add address=192.168.1.1/24 interface=ether？;
      • 等号后面为想要设置的地址，如果用的是软路由安装的必须和管理平台（如esxi）地址在同一网段；ether？为ros网口的设置。如不是初始安装需要先移除接口IP：ip address remove 0 （0为第一个网口ether1前的标识）

  2. 拨号上网（需要路由器来拨号的）

    • 2.1 interface 修改端口名称，wan口、lan口 ；
    • 2.2 ip - dhcp client - 增加一条 interface：后选择 wan 口；
    • 2.3 interface - 点击加号新增 - 选pppoe clint - general选项下 interface：后选择 wan 口 - dial out 选项下user：后填写宽带账号；password：后填写宽带密码 - 点OK
    • 注：use peer dns 前的勾不用打，如果要用服务商dns则要选上。拨号成功后IP选项下的address后新出现一条地址信息。
    • 注：修改ros后台IP地址 ip - address - address：后填写ros后台地址，如192.168.1.1/24；network：后为相应的网口地址，如192.168.1.0；interface：后选择lan口。

  3. 设置防火墙

    • 3.1 ip - firewall - NAT - 新增New NAT Rule - Action - Action:后选择 masquerade ；
    • 3.2 general 选项后 chain：后选 srcnat 最后应用确定。

  4. 设置DNS

    • 4.1 IP - DNS - Allow Remote Requests 前打勾 - Servers: 后填写需要填写的DNS服务器地址（比如：114.114.114.114） - 点OK
    • 4.2 在电脑端将DNS服务器设置为 ROS 服务器地址

  5. 设置网桥

    • 5.1 bridge - 点击加号，新增一条保存；
    • 5.2 bridge - Ports - 点击加号 - general interface：下选择网口；bridge：下选择刚才创建的网桥名称 - apply - ok 按照本方法逐一添加其他lan口（wan口不能加入）。

  6. 设置DHCP

    • 6.1 创建地址池：ip - pol -点击加号 - address：后面填入地址范围（如192.168.1.10-192.168.1.255），根据自己的网段需求设置，避开已设置的ip地址；
    • 6.2 ip - DHCP server - 点击加号新增 - interface：后选择网桥名称（如bridge1） - address pool：后选择刚建的地址池名称（如：dhcp_pool0）；
    • 6.3 DHCP server - network 选项 - 点击加号新增 - address：192.168.1.0/24 （填写自己的网段0/24不变）；网关gateway：后填写ros地址（如192.168.1.1）；子网掩码netmask:填24；dns servers 设置为公共dns地址（如114.114.114.114）；
    • 6.4 将电脑的手动网络配置改为自动。

  7. 双软路由设置

    • 7.1 ros上设置：将DHCP server - network 选项 里面的 网关gateway：后填写ros地址（如192.168.1.1）修改为旁路由 ip地址 （如旁路由为openwrt 地址为192.168.1.2）为192.168.1.2 其余不变；
    • 7.2 旁路由openwrt初始设置：如openwrt 还未设置 用 vi /etc/confige/network 进入网络设置 - 点键盘上 i 键 -鼠标移至 config interface‘lan’下option ipaddr 后地址改为自己需要设置的openwrt 后台地址（如192.168.1.2）- 点esc 退出编辑状态 - ：wq 保存退出 - reboot 重启。
    • 7.3 旁路由openwrt上设置：后台地址192.168.1.2登录 - 网络 - 接口 - 将wan口和wan6 删除（或在物理设置下将接口选为空的网口） - 编辑lan口 - 协议选静态地址；IPV4地址 192.168.1.2；子网掩码 255.255.255.0；IPV4网关选择ros后台地址（如192.168.1.1）；dns服务器选ros后台地址 - 关闭下面DHCP服务器。
